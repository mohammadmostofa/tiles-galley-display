#  Tiles Gallery
A modern web application for displaying and exploring a collection of tiles with dynamic
 filtering and detailed views.

---

## 🚀 Project Overview

**Tiles Gallery** is a responsive and user-friendly platform that showcases
 various types of tiles. Users can easily browse, search, and view detailed information about each tile.

---

## 🛠️ Tech Stack

* HTML
* Tailwind CSS
* Next.js
* Better Auth
* Hero UI
* Daisy UI

---

## ✨ Features

* 🔍 **Search & Filter**

  * Real-time filtering through a search bar
  * Quickly find tiles based on keywords

* 🧾 **Dynamic Tile Details**

  * Each tile has a dynamic ID
  * Click to view detailed information about a specific tile

* 🎨 **Modern UI Design**

  * Clean, responsive, and smooth UI using Tailwind, Hero UI, and Daisy UI

* 🔐 **Authentication**

  * Secure authentication using Better Auth

## ⚙️ Installation & Setup

```bash
git clone https://github.com/your-username/tiles-gallery.git
cd tiles-gallery
npm install
npm run dev
```

Open: http://localhost:3000

---

## 🌐 Deployment

Deploy easily using:

* Vercel
* Netlify

⚠️ Make sure environment variables (auth keys) are properly configured.

## ⚠️ Important Notes
